# Window Count

Currently only supports [MangoWM](https://github.com/mangowm/mango)

Shows window counts, optionally in a direction from the focused window

## Plugin

| Field | Value |
| --- | --- |
| ID | `aurelia/window_count` |
| Entries | Service: `service`; bar widget: `widget` |

## Requirements

Install `node`

* [node](https://nodejs.org/)

## Usage

Has no non visual behaviour

## Settings

| setting | type | default | description |
| --- | --- | --- | --- |
| `direction` | `select` | `"All"` | Whether to count all windows or ones only a certain direction from the focused window |
