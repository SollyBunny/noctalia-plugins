# CPU Clock Speed

A bar widget to show the current maximum CPU clock speed

## Plugin

| Field | Value |
| --- | --- |
| ID | `aurelia/cpu_clock_speed` |
| Entries | Bar widget: `widget` |

## Usage

Has no non visual behaviour

## Settings

| setting | type | default | description |
| --- | --- | --- | --- |
| `glyph` | `glyph` | `"cpu"` | The glyph to show on the widget |
| `show_glpyh` | `bool` | `true` | Whether to show the widget on the glyph |
